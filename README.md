# Student Management System

A simple command-line Student Management System built with Python and SQLite.

## Features

* Add new students
* View all students
* Search students by name
* Update student information
* Delete students
* Store data using SQLite

## Technologies Used

* Python 3
* SQLite3

## Project Structure

```text
student-manager/
│
├── db.py
├── main.py
├── menu.py
├── .gitignore
└── README.md
```

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run: python menu.py

A students.db file will be created automatically when the program runs.
